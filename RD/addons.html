Аддоны:

 

 Аддоны разархивировать в папку AddOns


[ Swatter ] - вывод сообщений об ошибках lua в более читабельном и подробном виде, чем стандартный интерфейс.

[ Accomplishment ] - аддон для автоматического поздравления членов вашей гильдии, при получении ими достижений.

[ Ace2 ] - это дополнительный аддон, который обеспечивает работоспособность других аддонов.

[ Ackis Recipe List ] - поможет вам с поиском нужного рецепта для профессии (на карте или на продаже у вендора).

[ Addon Control Panel (ACP) ] - аддон название которого говорит нам его суть.

[ ArenaHistorian ] - предназначен в основном для игроков, которые хотят отследить всё о всех состязаниях на арене.

[ Arena Spectator ] - дает возможность следить за играми на арене через арена спектатора (показывает все используемые там способности, а так же хп и многое другое).

[ AtlasLoot ] - содержит базу вещей/предметов,где,с какого босса/моба падает та или иная вещь.

[ Auctionator ] - Помогает при торговле. Ведет свою базу данных, в которой записывает цены лотов. Ведет контроль покупок и многое другое.

[ Bartender 4 ] — это аддон, полностью замещающий стандартные панели. Он предоставляет Вам возможность полностью отредактировать пользовательский интерфейс и увеличить количество панелей способностей на экране.

[ ChatBar ] - Этот аддон разработан специально для любителей пообщаться в чате. Он позволяет выбрать удобный режим общения.

[ CombatLogFix ] - Аддон, который поможет вам разрешить проблему с некоторыми ошибками комбат лога, что уменьшит шанс отказа в работе аддонов.

[ ComboPointsRedux ] - Показывает комбо очки накопленные на цели, отслеживает стаки "Смертельного яда".

[ Cooldowns ]- показывает цифрами время восстановления ваших способностей.

[ Deadly Boss Mods (или сокращенно ДБМ) ] — аддон, оказывающий огромную пользу в рейдах, т.к. показывает время до применения способностей босса.

[ DiminishingReturns ] - Аддон для wow 3.3.5следящий за диминишингом(временем действия на контролирующие способности.

[ Doom CooldownPulse ] - по истечению перезарядки заклинания отображает его вспышкой в центре экрана.

[ DrDamage ] - отображает реальный урон, который может нанести ваша способность с учетом вашей специализации, бафов, дебафов на целе, а так же вашей экипировки.

[ Enchantrix ] - показывает в тултипе что и с какой вероятностью получится из той или иной вещи после распыления/просеивания.

[ epgp ] - система распределения вещей в гильдии.

[ ErrorFilter ] - позволяет убрать сообщения об ошибках "Вы слишком далеко", "Вы не знаете как этим пользоваться" и т.д.

[ EventAlert ] - выводит иконки с таймерами ваших проков в любой точке экрана.

[ femaleannouncer ] - внесет в игру небольшие примочки в виде крутого звукового эффекта при убийстве других игроков.

[ Fishing Buddy ] – небольшой аддон для рыбалки в World of Warcraft.

[ Gatherer ] – Просто необходим для рудокопов, травников, рыболовов. Показывает места жил, травы, рыбы и др.

[ GearScore ] - это аддон показывающий уровень экипировки персонажа.

[ Gladius ] – аддон , для игры на аренах. Он отображает фреймы противников с жизнями, именами, их кд и др.

[ HealBot ] – это уникальный аддон, который предназначен для лечения всего рейда.

[ Healers-Have-To-Die ] - аддон, показывающий хилов противника на полях сражений (и не только).

[ HideChatButton ] - позволяет отключать чат.

[ HudMap ] - это аддон, который предоставляет вам информацию о положении энкаунтера, данные о дистанции, позиции членов рейда.

[ IceHUD ] - аддон для wow 3.3.5 добавляет дуги возле вашего перса, которые отображают здоровье, манну, энергию и т.д.

[ Icicle ] - PvP Аддон для wow 3.3.5 добавляющий над хп барами врагов, кулдауны на их заклинания.

[ Informant ] - показывал иконку вещи при тыке на линк.

[ InterruptBar ] – небольшой аддон, предназначенный для отслеживания кд противника.

[ LagBar ] - Показывает ваш FPS (количество кадров в сек.) и пинг с сервером.

[ Lorti UI ] - делает более темными панели способностей.

[ Losecontrol ]- аддон для wow 3.3.5 дополнение к стандартному интерфейсу отображает во фреймах все виды контроля.

[ Molinari ] - аддон для профессии, делает "Измельчение", "Просеивание", "Распыление" и "Взлом замка" чуточку легче, просто удерживайте клавишу ALT и выбирайте предметы напрямую.

[ Move Anything ] – аддон для World of Warcraft, который позволяет детально настраивать все элементы интерфейса в игре, а главное – перемещать их по экрану, как вам заблагорассудится.

[ MySales ] – ведёт статистику купли продажи на аукционе.

[ Nice Damage ] - аддон, который заменяет шрифт стандартного комбат текста.

[ Omen ] - Аддон выводит уровень угрозы мобов по отношению к вам в удобном виде.

[ OmniCC ] - показывает кулдауны/перезарядку предметов или спеллов.

[ OneBag3 и OneBank3 ] - обьединяют все сумки в одну.

[ Overachiever ] - добавляет в стандартное окно достижений несколько дополнительных вкладок.

[ Parrot ] - аддон из категории комбат текст. Parrot отображает ваш комбат лог текст в виде маленьких картинок с цветовым обозначением типа урона(лечения и поглощения) и его количеством.

[ PlateBuffs ] - отображает баффы/дебафы над головами игроков/ботов.

[ Postal ] – сделает интерфейс вашей почты просто классным. Ускоряет работу с почтой.

[ Power Auras Classic ] – это самый лучший и многофункциональный аддон, который вам позволит сделать любые графические сигналы о бафе/дебафе или количестве хп/маны/энергии/ярости или оповещающий о кд на спелы и многое другое.

[ Prat 3.0 ] - С его помощью можно изменить до неузнаваемости стандартный чат в WoW, преукрасить и добавить дополнительные функции для лучшего визуального восприятия выдаваемой информации.

[ Quartz ] - красивый и информативный каст бар (и не только).

[ QuestHelper ] — это всего лишь набор маршрутных точек для заданий с сортировкой по расстоянию до них.

[ Raid Roll ] - Делает проще раздел добычи между игроками. Записывает все броски игроков и обьявляет победителя.

[ RangeDisplay ] - Дисплей с указанием расстояния до цели.

[ Recount ] - показывает урон/отхил и многое другое (с модификациями показывает отхил+абсорб или чисто абсорб).

[ Sell-O-Matic ] - весьма полезный аддон WOW для каждого который помогает продать хлам.

[ SexyMap ] - более красивая миникарта.

[ SlideBar ] - Боковая панелька, в которой отображаются аддоны.

[ SnowfallKeyPress ] - уменьшает отклик клавиатуры (и добавляет анимацию [ SnowfallKeyPressA ]).

[ SoundAlerter ] - озвучивает на английском используемые противником способности.

[ SpellAlerter ] - добавляет картинку с используемой способностью противника.

[ Stubby ] - это небольшой аддон, который просто захватывает Lua ошибки.

[ Talented ] - упрощает создание ветки талантов (позволяет создавать шаблоны талантов, копировать ветку талантов других игроков).

[ TB_ArenaPoints ] - показывает сколько дадут ArenaPoints в команде на арене.

[ TellMeWhen ]- позволяет делать визуальные, слуховые и текстовые уведомлеения о восстановлении способностей, бафах/дебафах и т.д.

[ TidyPlates ] - Этот аддон видоизменяет в игре все фреймы наиболее привлекательные и информативные.

[ Titan Panel ] - лучшая статусная панель для ВоВ.Titan Panel показывает разную статусную информацию, такую как:время,деньги, количество слотов сумки, уровень и т.д.

[ TradeskillInfo ] - Аддон содержит всю необходимую информацию о рецептах и реагентах. Высвечивает Вендоров, у которых можно купить недостающие вам предметы. Также показывает цены реагентов.

[ WIM ] - выводит личные сообщение в отдельное окошко. Имеется звуковое оповещение, хранение истории. Поддерживается многооконный режим (для каждого собеседника свое окно чата), в аддоне WIM имеется возможность свернуть чат (автоматически сворачивается при входе в бой). Таким образом вы не пропустите сообщения от собеседника.

[ XLoot ] - полностью заменяет стандартное близзардовское окно WoW для лута, добавляя широкие возможности по его настройке.

[ X-Perl ] - заменяет фреймы рейда СТРА,стандартные иконки героя пати и целей, плюсы выгляет аккуратней.Более удобные и красивые 3-мерные портреты персонажей, рамки статистики,пати,рейда,бафов и т.п.
